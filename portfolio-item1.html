DropZone

Learning & Implementing

This project is a simple front-end web app where I utilized the 'Dropzone', and 'Timestamp' from the 'react' library for an easy going UX. Implementing html, TailwindCSS, and Javascript to create a nice simple, clean, crisp UI.

The 'DropZone' is located at the left of the screen. Where the user drags the file, and drops. Adds a caption, and presses 'post' where it`s stored to the backend database, and rendered to the browser in the center of the screen.
